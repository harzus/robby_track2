# robby_track2
Hardware
- Tracked robot based on DFRobotShop Rover V2 with wheel encoders and IMU (Bosch BNO055), Interface Arduino
- Realsense F200
- Kangaroo PC onboard 

Software: ROS kinetic with Ubuntu 16.04.
- Odometry: robot_localization
- 3D Mapping: rtabmap_ros
